Voltar para Elementos Químicos
Início Experimentos Terminologias Cientistas Tabela Periodica Vidrarias

Becker

Um recipiente cilíndrico usado para aquecer e misturar líquidos.

Erlenmeyer

Frasco de vidro de fundo cônico utilizado para aquecer líquidos e misturas.

Pipeta

Usada para transferir volumes precisos de líquidos.

Bureta

Tubo longo e estreito usado para medir volumes precisos de líquidos em titulações.

Balão volumétrico

Usado para preparar soluções com volumes específicos.

Funil de separação

Utilizado para separar misturas imiscíveis, como óleo e água.

Condensador

Equipamento usado para resfriar vapores e transformá-los em líquido.

Dessecador

Usado para armazenar substâncias que precisam ser protegidas da umidade.

Kitassato

Frasco de vidro usado em filtrações a vácuo, geralmente acoplado a um funil de Büchner.

Funil de Büchner

Utilizado em conjunto com o kitassato para realizar filtrações sob pressão reduzida.

Tubo de ensaio

Pequeno recipiente cilíndrico utilizado para reações químicas em pequena escala.

Balão de fundo redondo

Utilizado para reações químicas que requerem aquecimento uniforme.

Cápsula de porcelana

Recipiente resistente ao calor, usado para evaporação de líquidos.

Cronômetro

Instrumento essencial para medir o tempo em experimentos.

Vidro de relógio

Pequeno prato de vidro usado para pesar sólidos ou evaporar líquidos.

Espátula

Usada para manipular pequenas quantidades de substâncias sólidas.

Agitador de vidro

Haste de vidro usada para misturar soluções e líquidos.
